The Basic Language of The Web: HTML

Posted by Laura Jones on Monday, 21st May 2024

All modern websites and web applications are built on three fundamental technologies: HTML, CSS and Java Script. These are the languages of the Web.

In this post, lets focus on HTML. What is HTML and why you should learn HTML?

What is HTML?

HTML stands for Hyper Text Mark-up Language. It's a markup language that web developers use to structure and describe the contents of a webpage. (It is not a programming language.)

HTML consists of elements that describe different contents of a web page. They can be paragraphs, links, images, videos etc. Web browsers understang HTML and render HTML as web pages.

In HTML, each elemet is made of 3 parts:

  1. Opening tag
  2. Closing tag
  3. Actual content

You can learn more at MDN web documentation.

Why should you learn HTML?

There are countless reasons for learning the fundamental language of web. These are 5 of them:

  • To be able to use the fundamental web development language.
  • To hand-craft beautiful websites, instead of depending on tools like Wordpress or Wix.
  • To build web applications.
  • To impress friends.
  • To have fun 😜.
