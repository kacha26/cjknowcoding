What is linux?

To understand what is linux, one should be familiar with the following points:

  • Operating System The OS : An operating system is a collection of software that can manage hardware resources and provide an environment for applications to run.
  • The linux kernel: A kernel is an intermediary between the hardware and the software.
  • Distro: A linux kernel combined with other software becomes the full on operation system which is called a linux distribution or simply Distro.
  • GNU/Linux: The additional software that is required for the linux kernel to be a full fledged operating system is provided by the GNU project. Hence, GNU/Linux is the original operating system. And the distrubutions are the implementations or the different flavours of GNU/Linux each one with their own goals and focus of interests.

There are hundreds of linux distros which are curated softwares of GNU/Linux. You can use any distro to practice the concepts in this Tutorials. My personal favourite is Arco Linux

Kernel, Shell, Bash, terminal and prompt:

  • Kernel: Kernel is a computer program that forms the core of an opereating system that performs tasks like managing hardware and memory, scheduling tasks etc.
  • Shell: Shell is the default interface to the linux system. Shell exposes the OS's services to human users or other programs / applications. Shell takes the human commands and give them to OS to perform. It is the outer layer of the Kernel / OS, hence the name "Shell"
  • Bash: Bash is the most popular shell program. Bash runs pretty much on any linux distro.
  • Terminal: Terminal is a program that runs shell. Humans interact with shell. Shell the runs shell. Shell inturn gives commands to OS. And the OS performs the task requested by the human.
  • Prompt: Prompt is the basic information that is provided to the user, when shell is opened. It prompts the user to enter an input.

Normal users and Super users:

A Super user has full control over the OS and can perform any action.

Represented by # on prompt.

A Normal user can perform a subset of actions that a Super user can perform.

Represented by $ on prompt.
